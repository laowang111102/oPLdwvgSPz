# Python个人记账管理系统 python1225

## 项目概述

本项目是一款基于Python Django框架的个人记账管理系统。系统具备完善的收入与支出记账功能，以及相应的数据统计和图表展示。适用于个人财务管理，也可作为毕业设计、课程设计或计算机设计大赛的参赛项目。

## 技术栈

- **后端：** Python Django
- **数据库：** MySQL
- **前端：** HTML

## 功能模块

- **用户管理：**
  - 登录
  - 注册
- **记账管理：**
  - 收入记账
  - 支出记账
- **统计与图表：**
  - 收入支出统计
  - 图表展示
- **个人信息管理：**
  - 个人信息修改

## 系统角色

- 普通用户：可以注册、登录、管理个人记账信息

## 运行环境

- Python环境
- Django框架
- MySQL数据库

## 部署步骤

1. 配置Python环境
2. 安装Django框架
3. 配置MySQL数据库
4. 运行项目源码
5. 访问前端页面

## 目录结构

```
.
├── app                # 系统应用目录
│   ├── models.py      # 数据模型
│   ├── views.py       # 视图处理
│   ├── urls.py        # URL配置
│   └── ...            # 其他应用相关文件
├── db                # 数据库文件
├── media             # 媒体文件
├── static            # 静态文件
├── templates         # 模板文件
├── manage.py         # 管理脚本
└── requirements.txt  # 依赖列表
```

## 核心亮点

- **基于Django框架：** 稳定高效，易于维护
- **图表统计功能：** 直观展示财务状况
- **完善的用户管理：** 支持登录注册，保障信息安全
- **个性化开发：** 可根据需求进行定制修改
